# Overview

前出しじゃんけんは、手札が見える状態で戦う対戦ゲームです。

プレイヤーはあらかじめ手札を持ち、
その中からカードを選んで勝敗を競います。

特徴：

- 手札が公開されている
- 勝利条件を事前に設定する
- AIや外部プログラムと対戦可能（Advice API）
- 見た目をスキンで差し替え可能

このリポジトリでは、主に以下を公開しています。

- Advice API仕様
- スキンパッケージ仕様
- 公開向けドキュメント
